Hovamarine

Image slider

The Hovomarine is an unmanned amphibious vehicle, which combines the capabilities of a hovercraft and a submarine. It has potentially vast applications in the field of search and rescue, recovery of space modules from the sea and maritime security.

Team: Joby Yeldo
Guide: Prof.A.Venu Gopal
Funding: Rs.55,000

Updates

  • Patent application process is underway. Lakshya and NIT Warangal will jointly fund the patent application on a 80:20 basis
  • The Hovomarine is planned to be displayed in the NITW Centre for Innovation and Incubation

Smart Meter

Image slider

The Smart Meter is a device which will automate collection of data from domestic electric meters. The scope also include developing a software module to provide advanced usage analytics.

Team: Setlur Abhiram, Lokesh Chandra, Ravitheja, Rahul P
Guide: Prof.D.V.S.S Siva Sarma
Funding: Rs.20,000

Updates

  • Graphical User Interface to send SMS to the remote meter end GSM developed

Recycled Concrete

Image slider

Self Compacting Concrete using recycled aggregate is a research project to test usage of concrete from demolished structures to produce new concrete. As opposed to the current industry which uses such 'recycled aggregate' for pavemanet blocks and tiles, this research aims at producing concrete capable of bearing heavy loads.

Team: Sumanth Reddy
Guide: Dr.Rathish Kumar
Funding: Rs.56,000

Updates

  • Over 300 concrete casted and tested with various proportions of recycled aggregate
  • Project abstarct shortlisted for 3rd International Conference for Sustainable Construction Materials and Technologies SCMT3, Kyoto, Japan in August 2013
